How It Works

Get Started in 3 Simple Steps

Point Twilio to Caipher

Inbound calls hit our API where we return streaming TwiML that pipes audio directly to ElevenLabs.

Link the AI Agent

Caipher listens over WebSocket, runs your script, and posts intent + caller data to the AI Router.

Trigger Automations

GPT-4 routing books calendars, updates CRM, and sends SMS/email confirmations automatically.
